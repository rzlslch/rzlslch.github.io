end of 2025, new ssbg
04 December 2025 - code

Di penghujung 2025, ganti ssbg lagi.

ssbg itu standalone static blog generator, semacem jekyll atau 11ty. Buat bikin blog static tanpa harus bikin wordpress atau blogspot.

Jadi ssbg sebelumnya pake flutter, sekarang rewrite jadi nodejs. Dan bentuknya cuman script aja. Isinya cuman 1 file. Dibuild pake liquidjs dan tetekbengek nya. Kalo yang versi flutter kemaren ada GUI nya, ini cuman cli. Tapi bisa passing command jadi kalo mau bikin post baru tinggal ketik di terminal new post "title post".

Kok ganti lagi?

Bosen soalnya. Tapi dari sisi teknis juga, ssbg versi flutter ga enak dipake. Text editor bawaannya jelek. Mending nulis di vscode kemana-mana. Tujuannya dari awal emang buat pengganti jekyll yang ribet banget pake nya. Soalnya cuman butuh buat ngeconvert markdown jadi html, kalo built-in sama editor itu overkill banget rasanya. Dan yang pasti, lebih berat. Njalanin flutter tiap mau edit blog, hell no. Dibandingin cuman njalanin node script, jauh banget.

Btw banyak warna-warni sekarang. Utak-atik css.

Sekarang tampilannya cuman begini.

Butuh waktu 3 hari buat nge-rewrite code nya ke nodejs (actually, 3 hari itu maksud nya per hari nya 1-2jam dicicil malem2, biar dramatis aja dan keliatan lama), dan code nya pun masih spaghetti, ga beraturan. Satu file, yang isinya function bejibun dan tag register yang amburadul. But, hey, it works.

Semoga di akhir tahun 2025 dan kedepannya bisa ngumpulin niat buat nulis2 lagi, karena emang ternyata menulis itu merupakan salah satu kegiatan yang bisa merangsang otak, dan saya akhir2 ini lebih banyak doomscrolling dan ngerasa itu ga guna banget.

Udah gitu aja. Akhir kata, Wassalamualaikum wr. wb.
