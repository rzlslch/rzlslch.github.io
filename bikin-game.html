Dulu waktu SMA, ada temen yang punya hape nokia n90, yang layarnya bisa diputer2. Di dalem nya, ada game yang judulnya escape. Saya dan temen-temen dulu sering "lomba" maen escape pake hape nya temen saya itu, buat gede2an score.

Ini saya nyoba bikin game escape, tapi dengan low effort, javascript, web-game. escape

Saya build game ini pake phaser, dan bukan 1:1 adaptasi dari game n90 dulu, seinget saya aja. Asset nya juga ngambil dari template phaser. Mungkin nanti bakal saya update biar lebih asik, tapi sekarang yang penting jalan dulu.
