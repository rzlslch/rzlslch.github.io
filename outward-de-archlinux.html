Outward DE Archlinux [English]
08 December 2025 - code

One of the best game I've ever played on Steam, is Outward (Definitive Edition). I love this game so much I've created a youtube channel about it. Not so much contents, but it's the furthest I've ever done for a game.

So, last year my old computer died for good. And I've saved up enough money to built another one, so I bought:

  • Ryzen 7 5700G
  • ASUS PRIME B450M-A II
  • KLEVV DDR4 Value Series PC21300 2666MHZ 32GB (2x16GB)
  • XFX RX 6600 8GB SPEEDSTER SWFT 210
  • Super Flower Leadex III Gold 650 W

I decided to install linux on it (and I got all AMD, it should be great on linux), because it seems now I can enjoy my Steam library of Windows Games on linux, almost natively, using proton.

You see, my last gpu was GTX 1060 6GB, so I thought RX 6600 is at least an upgrade. And I got it for the same price as brand new GTX 1060 6GB many years ago, so I considered it a great upgrade. For the same price (even with inflation), it has newer chip and bigger memory.

I'm pretty disappointed. The last time I played Outward on my old computer using GTX 1060 6GB, I got around 60fps on average. And on my new computer, I only got around 70fps. Only around 10fps upgrade. I tried lowering my visual settings (I played on highest), fps are the same more or less. Well, at least the price is the same, and my GTX 1060 died anyway, so let's say it's the same GPU.

I played Outward Definitive Edition, and explore it's DLC. I traveled to one of its DLC Area, Antique Plateau, and start my adventure there.

But another problem hit me. You see, when you arrived at Antique Plateau, you start on Harmattan, a city. When I go outside the city, suddenly the fps drop, the game visual stutter badly. I don't have this problem before. Even with supposedly weaker GPU (being NVIDIA, on Linux, and older generation too, with smaller memory). Something must be wrong.

I tried another game. This time Dark Souls 3. I played DS3 and got 60fps. Sometimes lower. On my old computer, I remember higher fps than this. This is console level fps.

I look for solutions online. Many times searching, I forgot how I came up with it, but I read something about ASPM (Active State Power Management). On RX 6600, there's this setting that make the GPU use lower power mode, basically making its power limited.

I set my boot option to disable ASPM

sudo nano /etc/default/grub


...
GRUB_CMDLINE_LINUX_DEFAULT="... pcie_aspm=off"
...

Then I run sudo grub-mkconfig -o /boot/grub/grub.cfg to update my grub.

I rebooted my computer and start testing.

I started with DS3 first. The fps doubled. It's around 130 now. It works. On Outward should work too. I started Outward, and set the FPS limit to 999. It worked! My fps doubled too in Outward. And Antique Plateau no longer stutter. I continued to play with my newfound upgrades.

But that don't last long. Most of my playtime is flawless. But at some places, like Kazite Bandit Cave and Corrupted Tombs, my games starting to CTD (quit out, close to desktop). After many testing, I found a pattern, that everytime something with complex color and light shine (i don't know the term, something like blinding light) like Arcane Elemental is spawned, it's start to CTD.

This time, I started to test many proton version, and got a flawless gameplay again after changing my proton version to 8.0-5 for Outward.

My experience with this particular game isn't smooth for the early setup, despite protondb platinum rank. But now it's smooth sailing, I even made some build videos. I don't play much these days because life happens and I barely have any free time for gaming, but I can't wait until I can play this game again.

Afterwords note:

I have many games on my library. Most of them if not all, run perfectly with Proton. Even my non-steam games (and some of them are pirated, i'm sorry), works flawlessly. It just happens that my favorite game, the game that I poured so many hours into, is the one that needs a little hack to be able to be played flawlessly.

I don't know. I played more graphics intensive games without ever needing any setup (Red Dead Redemption 2, Ghost of Tsushima, Fallout 4, etc), why Outward being the one that needs special care?
